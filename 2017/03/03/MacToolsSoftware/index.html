Mac 常用工具类软件整理

发表于 2017-03-03   |   分类于 Mac   |  

看着别人高产似母猪，而我却。。一年憋出来一篇酱油贴，愧对江东父老啊。。。

  • iTerm2 原生终端替代品，配合 zsh 食用风味更佳。
  • Alfred 工作流效率神器。
  • MWeb 一款功能齐全的 MarkDown 编辑器。
  • Ulysses 一款国外的备受好评的 MarkDown 编辑神器，配合 Dropbox 或者 iCloud 使用最佳。
  • Spark 支持 iOS 和 macOS 的很优秀的邮件客户端。
  • TeamViewer 强大的远程桌面控制软件。
  • XMind / MindNode 思维脑图软件。
  • Dash 各种开发文档阅读。
  • ShadowsocksX 科学上网辅助工具。
  • PyCharm Python 编辑器。
  • Sublime Text 代码编辑器。
  • Charles 抓包工具。
  • Caffeine 防止电脑休眠工具。
  • VLC 视频播放工具，支持多种视频格式。
  • iFunBox 装 IPA 包工具，方便快速。
  • SimDirs iOS 模拟器沙盒路径读取工具。
  • Manico 快速切换 Mac 已开启应用的辅助工具。
  • Avast 免费的杀毒软件。
  • PlutoX 利用原生 API 全局代理的工具。
  • Sketch 强大的画图工具。
  • Pocket Mac/iOS 网页保存工具。
  • Popcorn-Time 开源的视频资源整理，观看工具。
  • 未完待续。。。
